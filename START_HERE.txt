================================================================================
                        STEP 1: INSTALL DEPENDENCIES
================================================================================

Choose ONE of these options:

OPTION A (Easiest - Windows):
    Double-click: install.bat

OPTION B (Command Line):
    npm install

This will take 1-3 minutes and download ~300-400 MB of packages.

================================================================================
                        STEP 2: START THE APP
================================================================================

After installation completes, run:
    npm run dev

Then open your browser to:
    http://localhost:5173

================================================================================
                        STEP 3: TRY THE DEMO
================================================================================

On the landing page, click "Try Demo" to see the app with sample data.

Or click "Get Started" to create your own study schedule.
